# Budget Tracker - Specification

## Project Overview
- **Name**: Budget Tracker
- **Type**: Expense/Income Tracking WebApp
- **Core Functionality**: Track expenses, income, categories, and visualize spending
- **Target Users**: Anyone who wants to manage personal finances

## Tech Stack
- Vue.js 3 + TailwindCSS
- localStorage for data

## Features
- Add income/expenses
- Categories (Food, Transport, Shopping, Bills, etc.)
- Monthly view
- Balance tracking
- Visual charts
- Budget goals
